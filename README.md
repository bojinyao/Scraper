# Web-Scraper
 BTC Web scraper made with mongoose db, node, and express.


## Getting Started:
 - Clone the repository
 - Open the cloned repo directory with terminal/cdm prompt
 - Run `npm run start` 
 - Open up `localhost:8080` in your browser
 
## Technologies Used:
- Cheerio
- Express
- Mongoose DB
- Handlebars
- BlueBird
- BootStrap
